About this Tool

The Singlish to Unicode Converter is a free Sinhala language conversion tool designed for Sri Lankans. When you type Sinhala words using English letters (Singlish), it automatically converts them into Sinhala Unicode.

How to Use

1. Typing in Singlish

Type words in Singlish into the "Type Singlish" area. For example, when you type "kohomada," it will be converted to "කොහොමද."

2. Selecting Output Format

You can choose the required font type (Unicode, FM Abaya, or ISI Font) from the "Output Format" section. Unicode is recommended for most modern websites and documents.

3. Copying Text

The converted Sinhala text appears in the "Unicode Result" section below. You can easily copy it using the "Copy" button.

Useful Features

  • Voice Recognition: You can type in Sinhala using your voice.
  • Import/Export: Import and export .txt or .pdf files.
  • Character Map: This helps you view the transliteration rules and character layout for Sinhala letters.
